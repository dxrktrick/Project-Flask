Información del Proyecto.

Desarrollar un Sistema Gestor de Base de Datos que lleve a cabo el registro de diversos productos relacionados con un sistema de ventas conceptualizado con 3 categorías (tecnología, accesorios y papelería), contando con diferentes tablas que mantengan la integridad de una base de datos relacional, es decir, que se organicen como: Empleados, Productos, Categorías, Proveedores, y Clientes; Agregando dos más como: Ventas, Facturación, y consecuentemente estas dos últimas tablas reflejando el sistema de canasta o punto de venta, interactuando con todos los datos, finalmente, todo este proyecto también desarrollado con IDE VS Code, y un framework llamado Flask (Pyhton y HTML), dando producto como un Punto de Venta a base de una página web.
